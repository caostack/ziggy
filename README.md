# Ziggy

A toy terminal text editor written in Zig.

## Features

- ✨ UTF-8 support
- ⌨️ Basic terminal editing
- 📝 File read/write
- 🎯 Gap Buffer data structure

## Keybindings

- `Ctrl+S` - Save
- `Ctrl+Q` - Quit
- `Ctrl+C` - Force quit
- Arrow keys - Move cursor
- Enter - New line
- Backspace - Delete

## Building & Running

```bash
# Build
zig build

# Run
zig build run

# Edit a file
zig build run -- your-file.txt
```

## Note

⚠️ **This is a toy project** for learning and demonstrating Zig programming. Not intended for production use.

## Requirements

- Zig 0.15.2 or later
